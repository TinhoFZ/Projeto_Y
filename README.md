Projeto Y é um projeto que visa ajudar professores no ensino da matemática. 
Atualmente possui as seguintes funcionalidades: 
Login e cadastro;
Visualização de categorias e módulos;
Marcar progresso no módulo (Parcialmente incompleto).

Funcionalidades planejadas:
Visualizar progresso no módulo;
criar seus próprios módulos personalizados.


Como usar:
Coloque o código em uma pasta;

Dentro da pasta do projeto:

py -m venv .venv |
.venv\Scripts\activate |
pip install Flask; mysql-connector-python; flask-bcrypt |
py app.py
